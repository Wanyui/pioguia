# Code of Conduct

## Nuestro compromiso

Nos comprometemos a mantener una comunidad abierta y respetuosa para todas las personas.

## Comportamiento esperado

- Trato respetuoso y constructivo.
- Critica tecnica centrada en el codigo, no en la persona.
- Colaboracion con buena fe.

## Comportamiento inaceptable

- Acoso, insultos o ataques personales.
- Publicacion de informacion privada de terceros.
- Cualquier conducta que genere un entorno hostil.

## Aplicacion

El mantenimiento del proyecto puede moderar, editar o eliminar contenido que no cumpla estas normas.

## Reporte

Para reportar incidentes, consulta `SECURITY.md` o abre un canal privado con el mantenedor.
